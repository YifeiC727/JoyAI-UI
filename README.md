# JoyAI UI Prototype

JoyAI AI 能力平台交互原型，包含首页分发和功能工作台两个核心页面。

## 页面结构

| 文件 | 说明 |
|------|------|
| `index.html` | 入口，自动跳转至首页 |
| `joyai-homepage.html` | 首页分发页（支持 A/B/C/D/E 五种视觉风格切换） |
| `joyai-ui-prototype-v3.html` | 功能工作台（顶部 Tab 分类 + 左侧子功能导航） |

## 功能分类

- **对话助手** — 文字助手 / 流式理解
- **视觉创作** — 即时生成 / 流式编辑 / 长视频生成 / 世界模型
- **语音交互** — 语音聊天 / 音色创作 / AI 配音
- **具身仿真** — 仿真引擎

## 资源目录

```
Assets/             Logo、背景图、模型卡片图
字体/               京东朗正字体（1.0 / 2.0）
```

## 本地预览

直接用浏览器打开 `index.html`，或启动任意本地静态服务器：

```bash
npx serve .
```

## 版本切换

工作台页面右下角提供 A/B 切换器，可在两种布局间对比：
- **A** — 左侧 Sidebar 导航布局
- **B** — 顶部 Tab 导航布局（当前版本）
